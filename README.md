# Selenium WebDriver 4

Exploring some of the new changes in Selenium WebDriver 4.

## Features

- Performing Actions 🖱️
  - Drag & Drop
  - Click & Hold
  - Double Click
- Chrome DevTools 🛠️
- Relative Locators ↔️
- Screenshots 📸
- Waits 🚦
- Handling Windows 🪟
  - New Tabs
  - New Windows
  - Minimize Window
- Browser Options 🖥️
